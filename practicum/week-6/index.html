In practicum 5 you created a react application, imagequiz, and deployed it to https://your-github-username.github.io/imagequiz/. In this practicum you extend your imagequiz application to have the following functionality:
  1. When users click on an image, they are navigated to a page (this page is a page in your application) that contains a quiz. The user then can take the quiz.
  2. Each quiz has six questions.
  3. Each question on the quiz has the following format: a picture and four options to choose from with only one of them being the correct answer. Each option contains a word supposed to be a name for the object in the picture. The user can only select one option in each question.
  4. At then end of the quiz the score of the user will be shown on the page. The user can reapet the same quiz or can go to the home page to start another quiz.
This practicum resembles the image quiz application of Cambridge Dictionary which is available at https://dictionary.cambridge.org/us/plus/quiz/image/. Have a look at the Cambridge Dictionary image quiz application to gain a better understanding of the practicum requirements.
For this practicum, you only need to have three quizzes. So on the homepage display at least three quizzes.

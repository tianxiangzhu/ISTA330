In this practicum you write a simple web application in React. You need to create a new repository and name is imagequiz. Do NOT use your ISTA330 repository for your development and deployment of this practicum. You need to create a separate repository for this practicum.

The web application should have the following pages:
  1. A login page
  2. A home page
The folder 'images' contain several images. The homepage of the application should list all the images. For each image it should show the image and under the image it should show the name of the image.
Deploy your react application to github pages. If deployed successfully then your react application will be shown in the iframe below.
Follow the following naming convention when you create your react application:
  1. The name of your github repository for this application should be imagequiz.
  2. Below this line, you find an iframe element. The address that you provide for the iframe src attribute should be https://your-github-username.github.io/imagequiz/ where your-github-username is the username with which you created your github account.
